Identificadores dos itens nas listas

Listas ordenadas

Podemos usar os seguintes identificadores nas listas ordenadas no parâmetro type, e também podemos determinar de que valor ela iniciará com o parâmetro start:

  • 1 - lista numérica, por padrão começa assim
  • A - letras maiúsculas
  • a - letras minúsculas
  • I - numerais romanos maiúsculos
  • i - numerais romanos minúsculos

Listas não ordenadas

  • disc - bola totalmente preta, por padrão começa assim
  • circle - disco com as bordas pretas e interior sem preenchimento
  • square - quadrados totalmente pretos

Lista de Definições

É como se fosse um dicionário, com os termos e suas descrições, mas sem demarcadores.

Toda lista de definições é criado com a tag dl (definition list), cada um dos termos é criado com a tag dt (definition term) e cada uma das descrições é criada com a tag dd (definition description).

Exemplo de lista de definições

HTML
Linguagem de marcação utilizada para criar o conteúdo de um site
CSS
Linguagem de marcação para a especificação de estilos dos sites
JavaScript
Linguagem de programação para deixar os sites mais interativos
